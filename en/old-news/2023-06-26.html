Latest news 26-06-2023

NuTyX 23.06.1 available with cards 2.6.96

Spiky and me are happy to announce the new version of NuTyX 23.06.1 and cards 2.6.96 .

What's new

With the invaluable help of Dan (MiyoLinux), we're offering a new custom ISO JWM with a new theme implemented by Dan.

The flcards tool has been refreshed and now offers an even simpler interface.

The packagekit package has been added, and a backend for cards have been added and is under development. Means soon it will be possible to install, remove, upgrade NuTyX packages from the discover/gnome-packagekit applications.

The mybld -s command now requires a package/collection as an argument.

The interview given to Linux Magazine is published in full on the site.

SysV in 3.06 and systemd in 253.0 .

The xorg-server graphics server version 21.1.8, the Mesa 3D library in 23.1.2, Gtk4 4.10.4 and Qt 6.5.1.

The nvidia driver is update to version 530.41.03 . The legacy driver 470.xx series in version 470.182.03 . The legacy driver 390.xx series in version 390.157 .

The python interpreter is updated to version 3.11.4.

The XFCE desktop environment is updated to version 4.18.3.

The MATE desktop environment is a 1.26.1 version .

The GNOME desktop environment is also updated to version 44.2

The KDE desktop environment is available in Plasma 5.27.6, Framework 5.107.0 and applications in 23.04.2.

Available browsers are: Firefox 114.0.2, Chromium 114.0.5735.90, Epiphany 44.3, etc

Many desktop applications have been updated as well like Telegram-desktop 4.8.4, Thunderbird 102.12.0, Scribus 1.5.8, Libreoffice 7.5.4.2, Gimp 2.10.34, etc.

Core NuTyX ships with Long Term Support (LTS) kernels: 4.14.319, 4.19.287, 5.4.248, 5.10.185, 5.15.118, 6.1.35 and the latest stable version 6.3.9 .

13 desktops are now available in: XORG, OPENBOX, LXDE, XFCE4, JWM, CDE, LXQT, ENLIGHTENMENT, MATE, CINNAMON, BUDGIE, GNOME, KDE5.

They are available on the download page.

A simple:

will make it.
