NuTyX 24.04.0 available with cards 2.7.4

Spiky and me are happy to announce the new version of NuTyX 24.04.0 and cards 2.7.4 .

The novelty of this version is undoubtedly the release of KDE 6. The new version of KDE based on Qt6 is now available on NuTyX.

Updated packages

SysV in 3.09 and systemd in 253.0 .

glibc in 2.39, curl in 8.7.1, python in 3.11.8, etc .

The xorg-server graphics server version 21.1.11, the Mesa 3D library in 24.0.4, Gtk4 4.14.1 and Qt 6.6.3.

  • The nvidia driver is update to version 550.67 .
  • The legacy driver 470.xx series in version 470.239.06 .
  • The legacy driver 390.xx series in version 390.157 .

The python interpreter is updated to version 3.11.8.

The XFCE desktop environment is updated to version 4.18.3.

The MATE desktop environment is a 1.26.2 version .

The GNOME desktop environment is also updated to version 45.3

The KDE desktop environment is available in Plasma 6.0.3, Framework 6.0.0 and applications in 24.02.1.

Available browsers are: Firefox 124.0.2, Epiphany 45.3, etc

Many desktop applications have been updated as well like Telegram-desktop 4.14.1, Thunderbird 115.9.0, Scribus 1.6.1, Libreoffice 24.2.2.2, Gimp 2.10.36, etc.

Core NuTyX ships with Long Term Support (LTS) kernels: 4.19.311, 5.4.273, 5.10.214, 5.15.153, 6.1.83 and 6.6.23 and the latest stable version 6.8.2.

Upgrade process

This time it's a bit different if you are on KDE version5. Please do as follow: 

sudo init 3
sudo cards remove kde5
sudo cards upgrade
sudo cards install kde6
sudo shutdown -y r 0
After reboot you should be able to loggin into Plasma KDE6.

For all the other ISO, a simple 

sudo cards upgrade
   shutdown -y -r 0
should make it.
