Latest news 13-05-2023

NuTyX 23.05.0 available with cards 2.6.7

Spiky and me are happy to announce the new version of NuTyX 23.05.0 and cards 2.6.7 .

New toolchain gcc 13.1.0, glibc 2.37 and binutils 2.40 .

SysV in 3.06 and systemd in 253.0 .

The xorg-server graphics server version 21.1.8, the Mesa 3D library in 23.0.3, Gtk4 4.10.3 and Qt 6.5.0.

The nvidia driver is update to version 530.41.03 . The legacy driver 470.xx series in version 470.182.03 . The legacy driver 390.xx series in version 390.157 .

The python interpreter is updated to version 3.11.3.

The XFCE desktop environment is updated to version 4.18.3.

The MATE desktop environment is a 1.26.1 version .

The GNOME desktop environment is also updated to version 44.0

The KDE desktop environment is available in Plasma 5.27.5, Framework 5.105.0 and applications in 23.04.0.

Available browsers are: Firefox 113.0, Chromium 113.0.5672.63, Epiphany 44.2, etc

Many desktop applications have been updated as well like Telegram-desktop 4.8.1, Thunderbird 102.10.1, Scribus 1.5.8, Libreoffice 7.5.3.2, Gimp 2.10.34, etc.

Core NuTyX ships with Long Term Support (LTS) kernels: 4.14.314, 4.19.282, 5.4.242, 5.10.179, 5.15.111, 6.1.28 and the latest stable version 6.3.2 .

A BASE ISO (proposed in command line interface edition only without graphical interface).

13 desktops are now available in: XORG, OPENBOX, LXDE, XFCE4, JWM, CDE, LXQT, ENLIGHTENMENT, MATE, CINNAMON, BUDGIE, GNOME, KDE5.

They are available on the download page.

Bugs fixes

Wrong keyboard layout in Xorg/Wayland if choosing bepo keyboard.

Concept mistake: The mybld command needs the cards.devel package.

The mybld command doen't check if sources are available when passing the new version as an argument.

Upgrade process

A simple: 

sudo cards upgrade
will make it.
