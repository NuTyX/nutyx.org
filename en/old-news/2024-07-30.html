NuTyX 24.07.0 available with cards 2.7.4

Spiky and me are happy to announce the new version of NuTyX 24.07.0 and cards 2.7.4 .

The new version of LXQt 2.0 based on Qt6 is now available on NuTyX.

Fixed

Upgrade process

This time it's a bit different if you are on KDE version5. Please do as follow: 

sudo init 3
sudo cards remove kde5
sudo cards upgrade
sudo cards install kde6
sudo shutdown -y r 0
After reboot you should be able to loggin into Plasma KDE6.

For all the other ISO, a simple 

sudo cards upgrade
shutdown -y -r 0
should make it.
