NuTyX 24.04.0 disponible avec cards 2.7.4

Spiky et moi-même sommes heureux d'annoncer la nouvelle version de NuTyX 24.04.0 et cards 2.7.4 .

La nouveauté de cette version est sans aucun doute la sortie de KDE 6. La nouvelle version de KDE basée sur Qt6 est désormais disponible sur NuTyX.

Mise à jour

Cette fois, c'est un peu différent si vous êtes sur la version 5 de KDE. Procédez comme suit: 

sudo init 3
sudo cards remove kde5
sudo cards upgrade
sudo cards install kde6
sudo shutdown -y r 0
Après redémarrage you devriez pouvoir vous connecter sur Plasma KDE6.

Pour les autres ISOs, la simple commande classique permet la mise à jour: 

sudo cards upgrade
sudo shutdown -y -r 0
